# Project 실행

## pnpm 설치 및 실행

```bash
pnpm install
pnpm run dev
```

## storybook 실행

```bash
pnpm run storybook
```

## ⚙️ Git Branch 전략

- main
- develop
- feature
- bug
- hotfix

## ⛓️ Git Commit 컨벤션

- feat: 기능 (feature)
- fix: 버그 수정
- docs: 문서 작업 (documentation)
- style: 포맷팅, 세미콜론 누락 등.
- refactor: 리팩터링
- test: 테스트
- chore: 관리(maintain), 핵심 내용은 아닌 잡일 등
